在js中，我们可以轻易的看到 function(){} ,{}这两种对象声明方式。

如果 
var obj = function(){
this.id =1;
this.name='misko_lee';
}
以及有

var obj2={
id:1,
name:'misko_lee'
}

obj 与obj2 都是object。但是他们是完全不同的

obj属于声明对象。声明构造对象的规则。
真正的对象要到 var testObj = new obj(); //此时才有真正的对象。

而obj2 是定义对象。obj2就是一个实实在在的对象。可 testObj是一回事。

因此，也就是说,
function(){}是对象工厂,new 出对象实体。{}则是直接造一个对象实体。
